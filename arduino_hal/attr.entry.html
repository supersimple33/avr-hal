Attribute to declare the entry point of the program

Exactly one entry point must be declared in the entire dependency tree.

#[arduino_hal::entry]
fn main() -> ! {
    // ...
}

The entry function must have a signature of [unsafe] fn() -> !.

This macro is a reexport of avr_device::entry. It is only available when the rt (runtime) feature is selected (it is by default).
